## üì± CATEGORY 2: SCANNING CLIPS

**What to film:** Hand with phone tapping/scanning the NFC sticker

### Clip Requirements
- **Duration:** 2-3 seconds each
- **Focus:** Phone screen and sticker
- **Movement:** Natural tap motion
- **Angle:** Profile view showing both phone and product

### 10 Variations to Film

#### 1. Top Approach Scan
```
Action: Phone enters frame from top, taps sticker
Angle: 45¬∞ side view
Tip: Slow, controlled movement
```

#### 2. Side Approach Scan
```
Action: Phone enters from left side
Angle: Direct profile
Tip: Keep phone parallel to surface
```

#### 3. Close-Up Tap
```
Action: Extreme close-up of tap moment
Angle: Zoomed on contact point
Tip: Focus on sticker and phone back
```

#### 4. Left Hand Scan
```
Action: Left hand holding phone, tapping
Angle: Right side profile
Tip: Natural grip, screen visible
```

#### 5. Right Hand Scan
```
Action: Right hand holding phone, tapping
Angle: Left side profile
Tip: Mirror of left hand version
```

#### 6. Different Phone Model #1
```
Action: Use iPhone if you have one
Angle: Standard profile
Tip: Shows compatibility across devices
```

#### 7. Different Phone Model #2
```
Action: Use Android if you have one
Angle: Standard profile
Tip: Demonstrates NFC universality
```

#### 8. Vertical Background Scan
```
Setup: Scanning on vertical surface (door)
Action: Phone taps sticker on door
Tip: Shows wall/door use case
```

#### 9. Horizontal Background Scan
```
Setup: Scanning on table/desk
Action: Phone taps sticker on flat surface
Tip: Most common use case
```

#### 10. Slow Motion Tap
```
Action: Same as any above, but in slow motion
Setting: 120fps or 240fps if available
Tip: Dramatic, emphasizes the "tap" moment
```

### Filming Tips

**Phone Visibility:**
- Show phone screen when possible
- Keep phone recognizable (Apple/Samsung logo)
- Clean phone case/back

**Tapping Motion:**
- Natural speed (not too fast)
- Confident approach
- Slight pause at contact point (0.5 sec)

**Camera Angles:**
- Profile view shows both elements
- Slight elevation for better view
- Don't obstruct sticker or phone
